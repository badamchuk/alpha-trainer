# Skill: start

Виконується на початку кожної нової сесії.

## Кроки

1. Прочитай `.claude/SNAPSHOT.md` — зрозумій де зупинились
2. Прочитай `CLAUDE.md` якщо потрібно освіжити контекст
3. Перевір поточний стан файлів якщо є незакрите завдання зі SNAPSHOT
4. Коротко доповіщай: що бачиш у SNAPSHOT, що готовий робити

## Відповідь після старту

Формат:
```
Завантажив контекст. Останній стан: [1 речення з SNAPSHOT].
[Якщо є незакінчене — "В процесі: ..."]
Готовий. Що робимо?
```
