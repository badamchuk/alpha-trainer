# Skill: finish

Виконується в кінці сесії або перед тривалою паузою.

## Кроки

1. Оціни що було зроблено за сесію
2. Оновити `.claude/SNAPSHOT.md`:
   - Розділ "Останнє оновлення" — сьогоднішня дата
   - Розділ "Що зроблено" — додай нові пункти
   - Розділ "В процесі / наступне" — що незакінчено або що планується
   - Розділ "Відомі проблеми" — якщо знайдено нові
   - Розділ "Важливі рішення" — якщо прийнято нові архітектурні рішення
3. Оновити `CLAUDE.md` розділ "Поточний стан" якщо суттєво змінився стан проекту

## Відповідь

```
Сесію завершено. SNAPSHOT оновлено.
Зроблено: [список]
Наступного разу: [якщо є]
```
