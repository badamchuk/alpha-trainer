# Commits

НЕ коммітити нічого якщо користувач явно не попросив.
НЕ пушити в remote ніколи — якщо тільки не явна команда "запуш".
НЕ використовувати --no-verify, --force, --no-gpg-sign без явного дозволу.

Коли користувач просить закоммітити:
- Staged + unstaged diff переглянь перед коммітом
- Не включай .env, ключі, логи, великі бінарні файли
- Повідомлення коміту — коротко суть змін, українською
