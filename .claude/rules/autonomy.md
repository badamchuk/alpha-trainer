# Autonomy

Роби самостійно, не питай зайвий раз.

## Роби без підтвердження
- Читання будь-яких файлів
- Редагування існуючого коду
- Запуск TypeScript перевірки (`tsc --noEmit`)
- Пошук по коду (`grep`, `find`)
- Встановлення пакетів (`npm install`)

## Питай перед тим як
- Видалення файлів або папок
- `git reset`, `git checkout .`, будь-які деструктивні git операції
- Запуск збірки (`npx expo run:android`) — бо займає 2-5 хв, треба впевнитись що зміни готові
- Будь-що що торкається production або публічних сервісів

## Якщо щось незрозуміло
Спробуй сам — прочитай файл, пошукай по коду. Не питай "де знайти X?" — знайди сам.
